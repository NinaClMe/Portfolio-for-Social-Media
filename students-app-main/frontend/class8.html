Command line interface - CLI

A command-line application, also known as a console application or command-line interface (CLI) program, is a software application designed to be operated primarily through a text-based command-line interface in a terminal or command prompt.
It is a text-based user interface that allows users to interact with a computer or software by typing commands into a terminal or command prompt. It provides a direct and efficient way to perform tasks, execute programs and manage system configurations using text commands rather than graphical interfaces.


Node.js and vscode

Node.js is a runtime environment that allows you to execute JavaScript code server-side. It is built on the V8 JavaScript runtime engine, the same engine that powers the Google Chrome browser.
It is an open-source, server-side JavaScript runtime environment built on the Chrome V8 JavaScript engine. It allows developers to execute JavaScript code outside of a web browser, enabling server-side scripting for building scalable and efficient network applications. Node.js provides a rich ecosystem of libraries and frameworks, making it popular for developing web servers, APIs and real-time applications.
IDE: Integrated Development Environment

Visual Studio Code (vscode) is a free, open-source source code editor developed by Microsoft. It provides a lightweight yet powerful environment for writing, editing and debugging code across various programming languages. VSCode offers features such as syntax highlighting, code completion, debugging support, version control integration and an extensive ecosystem of extensions to enhance functionality and customization for developers.


Dependencies

In the context of software development, dependencies refer to external modules, libraries or packages that a software project relies on to function correctly. These dependencies are typically third-party code or pre-built components that provide specific functionalities or services. They are used to enhance the capabilities of a project, avoid reinventing the wheel and promote code reuse across different applications.


Application ports

A port is a communication endpoint that is used by protocols to identify a specific process or service on a device

Application ports, often referred to simply as "ports," are numerical identifiers used to differentiate between network services or processes running on a computer. In computer networking, ports allow multiple applications or services to operate simultaneously on a single device by assigning each one a unique port number. Ports are part of the addressing information used in the transport layer of the Internet Protocol (IP) suite. They enable communication between different applications or services by specifying the source and destination endpoints of data transmission within a network. Commonly used port numbers include 80 for HTTP (web servers), 443 for HTTPS (secure web servers) and 22 for SSH (secure shell).
